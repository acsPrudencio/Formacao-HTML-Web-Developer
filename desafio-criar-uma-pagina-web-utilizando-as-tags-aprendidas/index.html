BLOCKQUOTE

Usada para indicar citações

H1,H2,H3,H4,H5,H6

Usada para representar titulos, quanto menor o valor de h(minimo de 1), maior o tamanho do texto

STRONG

Usada para marcar uma trecho de código como negrito

SMALL

Defini um texto com tamanho reduzido, geralmente é usada para comentários

MARK

Usado para destacar um texto: dio

HTML

Tag incial para a construção de uma página, todas as demais tags são criadas depois dela

HEAD

Tag que marca o cabeçalho da página, nela podemos definir o titulo que aparecerá na guia

FONT

A tag font foi usada no HTML 4 para especificar a fonte, o tamanho da fonte e a cor do texto.

ABBR

A tag abbr define uma abreviação ou um acrônimo. EX: JS

DEL

A tag define o texto que foi excluído de um documento.

SUB

Usado para representa base de caracteres: EX: log2 n

HR

Adiciona uma linha que preenche toda a tela na horizontal

UL

Usada para criar listas não ordenadas

OL

Usada para criar listas ordenadas

P

Usada para iniciar e fechar um parágrafo

U

Usada para sublinhar um trecho de texto

A

Usada para criar links

I

Usada para deixar um trecho de texto em itálico
